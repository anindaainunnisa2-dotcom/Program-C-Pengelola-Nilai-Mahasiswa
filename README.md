Program Pengolahan Nilai Mahasiswa (Bahasa C)

Penjelasan Program
Program ini merupakan aplikasi berbasis console yang dibuat menggunakan bahasa pemrograman C untuk mengelola dan mengolah data nilai mahasiswa dalam satu kelas. Program dirancang sebagai simulasi tugas asisten dosen dalam membantu proses pengolahan nilai akademik mahasiswa.

Dalam program ini, data mahasiswa disimpan menggunakan array satu dimensi dan array dua dimensi. Array satu dimensi digunakan untuk menyimpan NIM mahasiswa, sedangkan array dua dimensi (matriks) digunakan untuk menyimpan nilai Tugas, UTS, dan UAS. Jumlah mahasiswa dibatasi maksimal 30 orang sesuai dengan ketentuan pada soal.

Program menyediakan menu utama yang berjalan menggunakan perulangan do-while sehingga pengguna dapat memilih operasi yang diinginkan secara berulang. Operasi yang tersedia meliputi menampilkan seluruh data mahasiswa dalam bentuk tabel, menambahkan data mahasiswa baru, mengubah nilai mahasiswa berdasarkan NIM, serta menghapus data mahasiswa dengan cara menggeser isi array agar tidak terdapat data kosong.

Selain pengelolaan data, program juga melakukan pengolahan nilai menggunakan matriks, yaitu menghitung rata-rata nilai setiap mahasiswa, menghitung rata-rata nilai Tugas, UTS, dan UAS untuk seluruh kelas, serta menentukan mahasiswa dengan nilai rata-rata tertinggi dan terendah.Program ini juga menerapkan konsep fungsi rekursif, yaitu untuk menghitung nilai faktorial dari jumlah mahasiswa dan menampilkan deret bilangan Fibonacci sebanyak jumlah mahasiswa. Dengan demikian, program ini mengintegrasikan konsep array, perulangan, matriks, dan rekursif secara terpadu sesuai dengan tujuan praktikum Algoritma dan Pemrograman Lanjutan.
